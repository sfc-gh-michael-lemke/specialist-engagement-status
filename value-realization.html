Solution Overview

Specialist Engagement Status: Natural language triage for overdue engagement updates

A Cortex Code skill that classifies every specialist's engagement update status (Recent, Needed Soon, Needed Now) by querying a 30-minute refreshed materialized table — triggered by plain-language prompts from SE managers.

Business Challenge
Managers have no automated signal for stale specialist engagement logs
  • SE managers manually review CRM records to find specialists who have not logged a comment or activity recently — a time-consuming weekly ritual
  • No urgency ranking means overdue specialists are discovered reactively, often after forecast calls have already occurred
  • Querying engagement staleness required writing SQL against multiple Salesforce and SE Reporting tables
Snowflake Capabilities
Natural language skill powered by Snowflake data and Cortex Code
Cortex Code CoCo Skills Dynamic Tables Snowflake Tasks
Business Outcomes
Proactive engagement management replaces reactive CRM audits
⚡
Instant Status in Plain English Ask "who needs updates on Zaki's team" and get a ranked markdown table in seconds, no SQL required
🎯
Urgency-Ranked Output Needed Now specialists surface first, ensuring managers act on the highest-risk gaps before forecast calls
🤖
Always-Fresh Data Materialized table refreshed every 30 minutes via Snowflake Task ensures status is never more than half an hour stale
